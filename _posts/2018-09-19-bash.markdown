Bash script options
-------------------

See also <http://www.tldp.org/LDP/abs/html/options.html>

Commonly, 'set -eux' is used.

    #!/bin/bash
    set -eux

    # script goes here

This will set the script to fail on any error (e), fail on unset
variables (u), and to expand commands and display them (x).
